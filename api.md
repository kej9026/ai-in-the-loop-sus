## 1. TMDB (Movies) - 영화 정보
가장 풍부한 데이터를 제공합니다.

기본 정보: 제목(원제/번역), 줄거리(Overview), 개봉일, 러닝타임, 장르, 태그라인(슬로건), 포스터/배경 이미지 URL.
제작 정보: 예산(Budget), 수익(Revenue), 제작사, 제작 국가.
평가: 평점(Vote Average), 투표 수.
크레딧 (별도 호출):
Cast (출연진): 배우 이름, 배역 이름, 프로필 사진.
Crew (제작진): 감독(Director), 작가(Writer), 촬영, 음악 등 모든 스태프.
기타: 시리즈(컬렉션) 정보, IMDB ID, 성인물 여부.
## 2. RAWG (Games) - 게임 정보
게임에 특화된 상세 데이터를 제공합니다.

기본 정보: 제목, 설명(Description), 출시일, 업데이트 날짜, 메인 이미지/추가 스크린샷.
플랫폼 & 스토어: 출시 플랫폼(PC, PS5, Switch 등), 구매 가능한 스토어(Steam, PS Store 등).
개발/배급: Developer(개발사), Publisher(유통사).
평가: 메타크리틱 점수, RAWG 자체 평점, ESRB 등급(연령가).
분류: 장르, 태그(분위기, 특징 등), 시리즈 정보.
미디어: 예고편 영상(Clip), 관련 웹사이트, Reddit URL.
## 3. Google Books (Books) - 도서 정보
서지 정보를 중심으로 제공합니다.

기본 정보: 제목, 부제, Authors(작가), Publisher(출판사), 출판일.
상세 정보: 설명(Description), 총 페이지 수(PageCount), 카테고리(장르).
